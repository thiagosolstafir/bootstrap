Desenhado para todos, em qualquer lugar

Precisa de razões para amar o Bootstrap? Olhe adiante.

Feito para e por nerds

Como você, nós amamos construir maravilhosos produtods para a web. Nós amamos tanto, que nós decidimos ajudar pessoas como nós para tornar isto mais fácil, melhor e rápido. Bootstrap é construído para você.

Para todos os níveis

O Bootstrap foi desenvolvido para ajudar pessoas de todos os níveis—designer ou desenvolvedor, um grande nerd ou alguém iniciante. Use-o como um kit completo para iniciar algo mais complexo.

cross-tudo

Originalmente, construído somente para browsers modernos em mente, o Bootstrap se esforçou para incluir um porte para a maioria dos browsers (até mesmo IE7!) e, com o Bootstrap 2, tablets e smartfones tabmém

grid de 12-colunas

Sistemas de grid não são tudo, mas ter um durável e flexível na base do seu trabalho pode tornar o desenvolvimento bem mais simples. Use nossas classes de grid já inclusas ou pegue a sua escolha

Design Responsivo

Com o Bootstrap 2, nós nos tornamos totalmente responsivo. Nossos componentes estão escalados de acord uma variedade de resoluções e dispositivos para disponibilizar uma experiência consistente, custe o que custar

documentação como guia de estilo

Diferente de outras ferramentas de front-end, o Bootstrap foi desenhado primeiro como um documento de guia de estilos não só para documentar nossas funcionalidades, mas as melhores práticas com exemplo ao vivo e exemplos de código

Biblioteca crescente

Mesmo tendo somente 10kb (gziped), O Bootstrap é um dos mais completos kit de ferramentas de front-end que existe com dezenas de componentes totalmente funcionais prontos para serem usados

Plugins jQuery customizados

Quão bom seria um incrível componente de design sem ser fácil de usar, apropriadamente, e com interações extensíveis? Com o Bootstrap, você tem um construtor customizado de plugins jQuery para trazer seus projetos a vida

Construído com o LESS

Onde CSS puro falha, LESS acerta com excelência. Variáveis, aninhamento, operações e mixings em LESS faz o código CSS mais rápido e mais eficiêncie sem um mínimo de sobrecarga
